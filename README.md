# **URL Shortener**

A simple URL shortener application built using Django.

## **Features**

- Shorten URLs
- Store and retrieve shortened URLs
- Redirect users to the original URL when they access the shortened URL

## **Prerequisites**

- Python 3.x
- Django 3.x
- SQLite or any other relational database

## **Installation**

```

```
